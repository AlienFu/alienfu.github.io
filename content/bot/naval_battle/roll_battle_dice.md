# Roll Battle Dice

Humans do **NOT** roll any dice. Bot(s) roll:

**Calculate Dice**

| Dip. Ideas | Artillery Dice |                              |                                    |
| :--------: | :------------: | :--------------------------: | :--------------------------------: |
|     0      |       1        | +1 die per adjacent Bot ship | -1 die if Battle Action or Admiral |
|     1+     |       2        | +1 die per adjacent Bot ship | -1 die if Battle Action or Admiral |
